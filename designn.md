🎨 Dashboard Design Specifications1. The Layout StructureSidebar (Primary Navigation): A slim, translucent sidebar featuring high-quality icons for Dashboard, Missions (active quests), Levels (the saga map), SOP Library, Leaderboard, and Rewards.  Top HUD (Heads-Up Display): A fixed header displaying real-time stats:Energy/Stamina Bar: (e.g., 120/150) to limit sessions and prevent fatigue.  Currency/Points: Gold coins and XP (Experience Points) total.  User Profile: A small, elegant card showing the user’s rank (e.g., "Top 15%") and current designation.  2. The "Saga Map" View (Main Content)This is the heart of the "Candy Crush" experience, replacing the standard list of procedures.  Visual Path: A winding, cinematic trail through different environmental biomes representing SOP categories (e.g., a "Safety Island" or "Technical Trench").  Interactive Nodes:Node Style: Floating circular nodes with level numbers.  Visual States: Completed (Green with a checkmark), Current (Pulsing neon border), and Locked (Greyed out with a padlock icon).  Quest Tiering: The map is divided into logical tiers: Foundation → Essentials → Operational → Advanced → Master.  3. The "Operational Level" Detail PanelWhen a user clicks a node (e.g., Level 3), a sleek side-panel slides in:Objective List: Clearly defined tasks like "Identify hazards" or "Apply SOP in real scenarios".  Level Rewards: Visual icons for the XP, Points, and specific Badges (e.g., "New Badge") available for 100% completion.  Action Buttons: A primary "Continue Quest" button and a secondary "Practice Mode" button for safe-to-fail learning.  4. Achievement & Reward TrayLocated at the bottom of the dashboard, this section reinforces positive behavior.  Badge Carousel: Cards showing earned badges like "First Steps," "Quick Learner," or "Safety Champion".  Progress Trackers: Linear bars showing how close a user is to completing a milestone (e.g., "Scenario Master: 7/10").  🛠️ UI Tech Stack ComponentsFeatureImplementation ToolGlassmorphismTailwind backdrop-blur-md and bg-white/10Map PathFramer Motion Layout and SVG path-drawing animationsStat CountersReact countup for smooth number transitions on XP gain3D PreviewReact Three Fiber (R3F) for rotating badge/award models
